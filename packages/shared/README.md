# `deriv-shared`
> Responsible for the shared utilities and styles for all packages.


##Commands:
setup :
  ```sh 
  lerna bootstrap
  ```
serve : 
  ```sh 
  npm run serve
  ```
build : 
  ```sh 
  npm run build
  ```
test  :
  ```sh 
  npm run test
  ```

## Usage
```css
@import 'deriv-shared/style/shared.scss'
```

```js
import { toMomemt } 'deriv-shared/utils/date'
```
