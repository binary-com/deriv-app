## Pre-installation

-   node >= 14.17.1
-   npm >= 7.1.0

## Quick start

1.  **Install your dependencies:**

    ```sh
    npm ci
    ```

2.  **To build publish file:**

    ```sh
    npm run build
    ```

3.  **Libary usage:**

    ```js
    import P2P from '@deriv/p2p';

    <P2P />;
    ```
